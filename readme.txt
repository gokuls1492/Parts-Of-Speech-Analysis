How to execute:
To run a jar file, execute following commands on terminal. (Pass src & dest as arguments)
1.	Copy the jar file(POSCount.jar from POSCount\target\) 
2.	hadoop jar hadoop.jar POSCount.jar WordCount.WordCount.POSCount <path-input-files> <path-hdfs-destination>
3.	POS List file is hard coded. Available in
/user/gxs161530/mobyposi.i
4.	To view extracted files on hdfs:
•	hdfs dfs -ls /user/outputwiki/
•	hdfs dfs -cat /user/outputwiki/part-r-00000 (to open file)
Imp: Output folder should not exist before execution
